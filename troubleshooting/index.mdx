Log errors, dead ends, and solutions here so you don't solve the same problem twice.

## How to log an issue

```markdown
### [Short title] — YYYY-MM-DD

**Environment:** OS, versions, tools involved

**Problem:** What went wrong and any error messages.

**Cause:** Why it happened (if known).

**Solution:** Steps that fixed it.

**Links:** Related docs or threads.
```
