This site is a personal knowledge base built with Mintlify. Use it to record what you study, problems you solve, and resources you want to keep.

## What you'll find here

- **Study** — Structured notes on topics like Isaac Sim, Isaac Lab, and robotics
- **Troubleshooting** — Error logs and fixes you've figured out
- **References** — Links, papers, and docs worth revisiting
- **Personal** — Daily log, snippets, and portfolio

## How to add content

1. Create or edit an MDX file in this repository.
2. Add the page to `docs.json` if you want it in the sidebar.
3. Run `mint dev` locally to preview changes.

<Tip>
  Start with [Study](/study) if you're learning something new, or jump to [Troubleshooting](/troubleshooting) when you hit a wall.
</Tip>
