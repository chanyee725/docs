This is a personal knowledge base for recording what you study, problems you solve, and resources you want to keep. Built with Mintlify.

## Sections

| Section | Purpose |
| ------- | ------- |
| [Study](/study) | Notes on Isaac Sim, Isaac Lab, robotics, and more |
| [Troubleshooting](/troubleshooting) | Error logs and fixes you've figured out |
| [Refs](/references) | Links, papers, and docs worth revisiting |

## How to use this site

1. Pick a section from the sidebar that matches what you're doing.
2. Edit the matching MDX file in this repository.
3. Preview locally with `mint dev`.

<Tip>
  Learning something new? Start in [Study](/study). Stuck on a problem? Check [Troubleshooting](/troubleshooting) first.
</Tip>
